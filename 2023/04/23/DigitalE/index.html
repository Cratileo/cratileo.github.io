1.电路基本知识

1-1. 重触发单稳态电路(Retriggerable one-shot circuit)
电路在触发时会存在一个“暂稳态”。对于不可重复触发电路，在进入暂稳态期间，触发脉冲对电路工作无影响，只有暂稳态结束后才能接受脉冲影响。而可重触发电路则没有这一限制，在暂稳态期间也可以接收脉冲信号，当接收到脉冲信号后，立即对电路进行重触发，使暂稳态延迟，直至触发脉冲的间隔超过单稳态输出脉宽，电路才返回稳态

例如某可重触发电路的脉宽(pulse width)为10ms，在被触发3ms后，立即输入另一脉冲信号，则此时的最近的输出脉冲(output pulse)将会推迟3ms，即13ms

1-2. 有限状态机(Finite State Machine)
在这里只需要了解以下知识(应该吧)
Moore model: 该状态机的输出仅与当前状态有关
Mealy model: 该状态机的输出不仅与当前状态有关，还与输入有关

1-3. 奇偶校验(Even odd parity)
奇偶校验一般选择在LSB位加校验码，校验规则是：
奇校验使数据流中存在奇数个1
偶校验使数据流中存在偶数个1

1-4. 二进制/十进制快速转换
整数应该都很熟悉了，这里重点谈论小数
注：这里不涉及float和double类型浮点数的转换（反正也不考）
将十进制数转换为二进制小数，方法是对小数位不断乘2，然后取出整数位，继续这一步骤，直到小数位为0

栗子: (5.8125)
0.8125x2 = 1.625 -> 1
0.625x2 = 1.25 -> 1
0.25x2 = 0.5 -> 0
0.5x2 = 1.0 -> 1
end,结果为101.1101

2.组合逻辑电路

2-1. 锁存器(Latches)
锁存器有R-S锁存器，D锁存器，J-K锁存器等多种，实现有与非门和或非门两种，对于与非门锁存器，其信号是active-LOW的，而或非锁存器是active-HIGH的

D锁存器和J-K锁存器的区别就是D锁存器只有一个输入端，而J-K锁存器有两个(触发器好像也一样)

2-2. 触发器(Flip-flop)
触发器在这里就不过多提及了，注意触发信号是由时钟上沿触发(positive)还是下行触发(negitive)，后者与前者在器件上的不同在于后者在时钟信号输入端加了非门

2-3. 二/十进制(BCD/DEC)转换器
对一般的BCD/DEC转换器，其一般存在4个输入端(A0-A3)对应四位二进制数，表示0-9，而输出端则是0-9十个端口对应十进制输出。每个十进制输出端都与其相应二进制表示对应，一个简单的栗子帮助理解：

2-4. 多路复用选择器(Multiplexer, MUX, Date selecter)
多路复用选择器一般存在n个S端(控制端)，和k个D端(选择端)。与BCD/DEC转换器的输入方式类似，也是由输入端按MSB到LSB构成一组二进制数表示，输入信号表示的二进制数转换为十进制后即对对应D端口信号进行选择输出，选中的那一路信号作为输出信号

2-5. 数据分配器(Demultiplexer, DEMUX)
是MUX的逆操作，即将单路数据分配到多路。有1路Data Input输入和2路选择信号输入，在此类模型中，将拥有4路D0-D3输出。和MUX的逻辑相仿，S端输入什么二进制信号时，数据输入端的信号就将从对应的输出端流出。
当然，DEMUX也可以进行扩展

2-7. 555定时器(The 555 timer)
对于555定时器，有以下公式：

充电时间t1 = 0.693*(R1+R2)*C1
放电时间t2 = 0.693*R2*C1
脉冲宽度Tw = 1.1*R1*C1
周期T = t1 + t2 = 0.693(R1+2*R2)C1
频率f = 1/T = 1.44/((R1+2*R2)C1)
占空比t1/T = (R1+R2)/(R1+2*R2)

公式列完了，来讲一点原理

图中所示的555定时器是一种无稳态电路，即OUT输出电压将会在自动在高低间变化。对于C2，一般总是为一个0.01uF的电容(理论上该路也可以悬空，但不建议这么做)

对555定时器而言，在其触发信号的一个周期内，输出信号会经历从充电到放电的全过程。一般在触发信号下沿时开始充电，而在另一触发信号下沿时放电结束。555定时器的内部电路是不支持重触发特性的。

2-8. 计数器(Counter)
2-8-1. 异步计数器(Asynchronous Counter)
异步计数器的时钟端是不相连在一起，而是串行连接的，而同步计数器则是共用一个时钟信号。这样就带来了以下区别

1.同步计数器在外部信号到来时触发器同时翻转；而异步计数器的触发器为串行连接，工作频率较低
2.异步计数器输出状态的建立，要比时钟慢一个传输时间，容易存在逻辑冒险


假定有n个触发器组成的异步计数器，其中单个触发器的延迟为10ns，则该计数器的传播延迟(Propagation delay)tp = n*10 ns,最小周期T = tp，最大时钟频率(maximum clock frequency)f = 1/T

2-8-2. 分频器与模n计数器

上图所示的微集成电路即为模n计数器，其C端输入同时也作为分频器使用。对该计数器而言，其模即为计数器上标识的数字，而时钟信号在接入C端口后也将会减少n倍，对多个计数器串联组成的电路而言，总模数等于各计数器模数和，总时延(tp)为各计数器之和，最小周期和最大时钟频率参照异步计数器。

栗子：对上图的计数器，总模(modulus)为200(= 4*10*5)，计数器输出频率(output frequency)为20Khz(= 4Mhz/4/10/5),假定每个边缘J-K触发器的时延为10ns，则最大时钟频率f = 33.3 MHz

3. 计数器设计

几个名词解析
Excitation equation: 激励方程(时序电路中触发器的输入方程，“激励”嘛，就像基电那个激励源)
Transition equation: 次态方程(可以理解为以Qn为自变量，Qn+1为因变量的方程，指出状态变化间的关系)
Characteristic equation: 特性方程（就是逻辑器件本身的端口特性方程，如下图）
